# Project Name
A Django REST e-commerce backend built with the philosophy that _pretty is common, robust is rare_, focusing on performance optimization, scalability, testing, and production-grade architecture.

---

## Tech Stack

- Python
- Django
- Django REST Framework
- PostgreSQL

---

## Setup

Clone the repository:

```bash
git clone https://github.com/ayush-logs/robust-ecom-backend.git
cd robust-ecom-backend
```
